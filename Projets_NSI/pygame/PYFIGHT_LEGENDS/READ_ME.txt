PYFIGHT LEGENDS !
_______________________________________________________________________________
Jeu Python avec Pygame 
_______________________________________________________________________________

Ce jeu peut être exécuté via Thonny avec PYFIGHT_LEGENDS.py. Il utilise le
module : module.py et la librairie pygame, ainsi que des images et autres
ressources (présents dans le dossier assets).
Le dossier doit être à côté du programme Python.

Si vous exécutez PYFIGHT_LEGENDS.exe, aucun fichier annexe n'est nécessaire.
_______________________________________________________________________________

Deux versions du code sont disponibles :
1. Version originale avec le code originel  
2. Version avec l'implémentation pour la création de l'exécutable
_______________________________________________________________________________
But du jeu
Faire les quêtes disponibles en bas à droite de la map,
puis battre le boss final accessible à partir du niveau 4.

Pour vous aider, des items sont disponibles à l'achat dans le shop.

Objets disponibles :
-> Potion soin ~ restaure les PV du joueur dans la quête en cours. Prix : 5
-> Livre ~ augmente l'attaque du joueur dans la quête en cours. Prix : 5
-> Potion de speed ~ augmente la vitesse de déplacement du joueur. Prix : 5

Pour accéder au menu de démarrage, appuyez sur CTRL + i.

Déplacement : Flèches (par défaut).
La touche de déplacement est modifiable dans les paramètres : bouton en haut à
gauche.

Le monstre présent sur la map inflige des dégâts. Si vous mourez, toute votre
progression est perdue.

Aucune sauvegarde du jeu n'est disponible.

Un easter egg sur le skin est à découvrir si vous terminez le jeu.
